This course introduced subagents as a way to delegate tasks in longer Claude
Code sessions to isolated assistants that work in a separate context window and
return only the information needed, keeping the main conversation focused. It
covered how subagents work, how inputs flow in and summaries come back, and
creating custom subagents with the /agents command for workflows such as code
review and documentation. It also covered patterns that make subagents reliable,
including structured output, obstacle reporting, and limited tool access, along
with when delegation is worth it and the anti-patterns to avoid.
