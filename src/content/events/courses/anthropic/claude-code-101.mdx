This course introduced Claude Code as an agentic AI coding tool and how it
differs from chat-based assistants. It explained how the agentic loop, context
window, tools, and permissions work together, and how to prompt effectively
using approval mode, auto-accept, and Plan Mode within the Explore, Plan, Code,
Commit workflow. It also covered customising Claude Code for real projects,
through a CLAUDE.md file for persistent memory, subagents for delegating tasks,
MCP servers for connecting external tools and data, and hooks for deterministic
control.
